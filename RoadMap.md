프로젝트 로드맵

역할 : 
건영 - 메인 디랙터
윤택 - 시니어 개발자
진 - 디자이너, 잡무

디자인 할 것 <- 기본 타일(도로, 벽, 방, 가구 등), 메인 캐릭터(), 적 캐릭터



-- 1단계 -- 


1. 시작 맵(주택가)을 만들어야겠어! (맵 자동 생성) <- [건영 (갈 수 있는 곳, !) 두 개의 타일(검정, 흰색)]

2. 맵 -> 이동 관련 -> nevigation(유니티 내비게이션)

3. 전투 시스템(player, enemy 공통 클래스 만들기, status, 피격 판정) <- [홍윤택]

4. 적 캐릭터 만들기(AI - nevigation)



-- 2단계 -- 


1. 좀비 스폰 및 웨이브 시스템 - 레벨 디자인 

2. 무기 시스템(아이템)

3. 기타 생존, 파밍 관련 요소




-- 3단계 -- 


1. 세이브 로드(캐시 저장) 

2. 탐험(맵 이동, 퀘스트 등)

3. 제작

4. 아이템 자동 스폰
